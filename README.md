# Project Name: 
-Keen Keeper Project

## Description
KeenKeeper is a personal relationship management tool that helps users nurture and maintain meaningful connections. By tracking interactions and analyzing engagement patterns, it ensures that important relationships are consistently cared for and never overlooked.

## Tech Stack
- Next.js
- Tailwind CSS
- Recharts
- Daisy UI
- Lucid Icon
- React Icon

## Features
### Friendship Tracking
-Easily understand the state of your relationships at a glance with clear status indicators like On Track, Almost Due, and Overdue.

### Interaction Statistics
-Visual dashboards break down how you communicate—calls, texts, or video—making your interaction habits easy to understand.

### Timeline View
-A clean, chronological feed lets you scroll through past interactions and stay aware of your connection history.

### States
-This States show you as a graph or charts according to friends data.
